# Caducus Worldbuilding

Welcome! This site is generated from Markdown files in this repository.

## Quick Start

- Browse canonical content under **Canon**.
- Use **Templates** when creating new entries.
- Keep metadata and IDs stable to make AI-assisted editing reliable.

## AI-Friendly Authoring Tips

1. Use explicit headings (`## Overview`, `## Canon Facts`, etc.).
2. Keep one concept per file.
3. Add stable IDs in frontmatter.
4. Record timeline events with exact dates.
